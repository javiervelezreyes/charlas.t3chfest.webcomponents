Mas allá de un mero secuenciador de transparencias, este componente incluye una linea de tiempo con información sobre eventos y acciones que se lanzan discrecionalmente a medida que avanza la reproducción. Los eventos se disparan para que una arquitectura de escuchadores reaccione. Aunque en este ejemplo dicha arquitectura no está proporcionada puede consultar los ejemplos de wc-speech para información al respecto. Las acciones por su parte operan sobre la reproducción invocando alguno de los métodos de negocio del componente con sus respectivos parámetros. Notará que el comportamiento de la reproducción no es, en este sentido, continuo. Siéntase libre de alterar el fichero JSON de timeline para probar sus efectos.
